# Yokoso

Yokoso is a platform or CMS for Developer Relations. You can launch Developer's site quickly for your service.

## Architecture

- node.js
- React x Cjsx x Server side rendering
- Express
- Coffee Script
- gulp
- MongoDB

## Usage

```
$ gulp start-mongodb &
$ gulp
$ open http://localhost:3000/
```
